Theory

Fibonacci Search is an efficient search algorithm based on the Fibonacci sequence. It is similar to binary search but divides the array into parts using Fibonacci numbers, making it advantageous in scenarios where division is costly.

Steps:

  • Uses Fibonacci numbers to split a sorted array.
  • Efficient for sorted datasets and hardware where addition is faster than division.
  • Runs in O(log n) time.

Advantages

  • Efficient for sorted arrays.
  • Avoids division, useful in low-level hardware.
  • Educational for understanding alternative search strategies.

Disadvantages

  • Requires a sorted array.
  • More complex than binary search for general use.
  • Less commonly used in modern applications.

Algorithm & Pseudocode


// Fibonacci Search Algorithm
function fibonacciSearch(arr, key):
    n = length(arr)
    fibMMm2 = 0
    fibMMm1 = 1
    fibM = fibMMm2 + fibMMm1
    while fibM < n:
        fibMMm2 = fibMMm1
        fibMMm1 = fibM
        fibM = fibMMm2 + fibMMm1
    offset = -1
    while fibM > 1:
        i = min(offset + fibMMm2, n-1)
        if arr[i] < key:
            fibM = fibMMm1
            fibMMm1 = fibMMm2
            fibMMm2 = fibM - fibMMm1
            offset = i
        else if arr[i] > key:
            fibM = fibMMm2
            fibMMm1 = fibMMm1 - fibMMm2
            fibMMm2 = fibM - fibMMm1
        else:
            return i
    if fibMMm1 and arr[offset+1] == key:
        return offset+1
    return -1

Step-by-Step Explanation

The Fibonacci Search calculates the probe index using Fibonacci numbers and compares the value at that index with the search key. Based on the comparison:

  • If the probe value is less, move the offset to the right and reduce the range.
  • If the probe value is greater, reduce the range on the left.
  • Repeat until the key is found or array is exhausted.

Real-Life Usage

  • Used in hardware and embedded systems where division is costly.
  • Good educational example for alternative search strategies.

Summary

Fibonacci Search is a unique search technique based on the Fibonacci sequence. It is fast for sorted arrays and uses addition/subtraction instead of division. Understanding it provides insight into alternative algorithm design techniques.
